# Simplify the following big O expressions as much as possible:
# 1. O(n + 10)
answer: O(n)

# 2 O(100 * n)
answer: O(n)

# 3 O(25)
answer: O(1)

# 4 O(n^2 + n^3)
answer: O(n^3)

# 5 O(n + n + n + n)
answer: O(n)

# 6 O(1000 * log(n) + n)
answer: O(n)

# 7 O(1000 * n * log(n) + n)
answer: O(n log n)

# 8 O(2^n + n^2)
answer: O(2^n)

# 9 O(5 + 3 + 1)
answer: O(1)

# 10 O(n + n^(1/2) + n^2 + n * log(n)^10)
answer: O(n^2)
